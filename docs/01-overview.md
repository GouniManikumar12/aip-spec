# 01 · AIP Overview

The Agentic Intent Protocol (AIP) defines how AI platforms, operators, and brand agents exchange commercial intent, govern participation, support delegation, and settle verified outcomes. This chapter summarizes the protocol's core goals: interoperability, transparency, verifiability, and privacy-safe execution for AI advertising and agentic commerce.

- **Interoperability:** JSON-based payloads with explicit versioning keep integrations predictable.
- **Extensibility:** Optional sections allow experimentation without fragmenting the ecosystem.
- **Safety:** Every transport includes authentication, encryption, and replay protection requirements.
